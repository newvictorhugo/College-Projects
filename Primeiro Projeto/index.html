Questionário sobre CSS (Folhas de Estilo em Cascata)




Perguntas frequentemente feitas por professores de programação:



1. O que significa CSS?

(Cascading Style Sheet) é uma linguagem de estilização de uma página web, aplicado diretamente nas tags HTML.


2. Cite 2 características importantes quando se usa CSS?

Saber que CSS se usa para alterar a posição, cor e fonte do texto. Quando for usar um estilo de comando, priorize aquele estilo, para a linha de comando ficar mais limpa.


3. Quais as possíveis formas de vincular o CSS em um documento HTML? Cite e exemplifique

Temos três estilo de utilização para CSS, temos o estilo incorporado, estilo importado e estilo inline. O estilo incorporado é usado quando não existe um padrão comum aos vários documentos de um site.

Por exemplo:

<html>
<head>
ㅤ<title></title>
ㅤ<style type="text/css"> <!-- Aqui se inicia o processo incorporado, ao invés de digitarmos o código que queremos dentro da pasta css, incorporamos esse mesmo código na pasta html -->
ㅤㅤh1 {text-align: center; color: blue}
ㅤㅤp {font-family: serif}
ㅤ</style>
</head>
<body>
ㅤ<h1>Página sem formatação de estilos</h1>
</body>
</html>


No estilo importado, a definição é feita num arquivo separado e importado na página.

Por exemplo:

<html>
<head>
ㅤ<title></title>
ㅤ<link rel="stylesheet" href="style.css" type="text/css"/> <!--Aqui foi importado a pasta css dentro da pasta html-->
</head>
....


No estilo inline, a definição de CSS é feita diretamente na tag HTML com o uso do atributo style.

Por exemplo:

<html>
<head>
ㅤ<title></title>
ㅤ<style type="text/css"> <!-- Aqui se inicia o processo incorporado, ao invés de digitarmos o código que queremos dentro da pasta css, incorporamos esse mesmo código na pasta html-->
ㅤㅤh1 {text-align: center; color: blue}
ㅤㅤp {font-family: serif}
ㅤ</style>
</head>
<body>
ㅤ<h1>Página sem formatação de estilos</h1>
</body>
</html>



4. Na definição do CSS o que representa a criação de uma regra no seletor?

Seletor define em quais elementos um conjunto de regras CSS se aplica. O que representa a criação de uma regra é o conjunto de elementos que são selecionados pelo seletor.

Por Exemplo:

h3{
ㅤ text-align: center;
ㅤ backgroud-color: #F8FCEF;
}

"h3": Seletor
Dentro da chave{ }: Regra do seletor


5. Como se comenta uma linha na definição do arquivo CSS?

Para comentar uma linha dentro de arquivo CSS se usa /* Texto */

Por Exemplo:

h3{
ㅤ text-align: center; /*Comentário sobre a linha de comando*/
ㅤ backgroud-color: #F8FCEF;
}


6. Quais são os delimitadores de início e fim na definição de uma regra do CSS?

Os delimitadores de inicio e fim são feitos pelas "chaves" {Regras}. Será dado o valor do seletor, em seguida abrirá as chaves ({), digitará a regra no seletor e fechará as chaves (}). Visualize o exemplo da questão 5.


7. Quando o ; será utilizado?

O ; é usado dentro do arquivo CSS para dizer ao programa que a leitura terminou naquela linha e que ele deve passar para a próxima. Visualize o exemplo da questão 5.


8. O que são valores múltiplos para serem usados nas regras CSS?

Valores multiplos, no CSS, são valores dentros das regras dos seletores, mas simplificados.

Por Exemplo:

.div {
ㅤ margin-left: 100px;
ㅤ margin-right: 60px;
ㅤ margin-top: 50px;
}

Dessa forma repetimos a mesma propriedade "margin", dando valor para cada lado, assim o código fica muito extenso. Não está errado, mas também não está da melhor forma.

.div {
ㅤ margin: 50px 60px 0 100px
}

O CSS disponibiliza atalho para você inserir valores em uma única propriedade seguindo uma ordem e um padrão.
Nessa propriedade você tem 4 maneiras de declarar múltiplos valores, veja:

● Para 1 valor: todos lados tem o mesmo valor.
● Para 2 valores: o primeira valor se aplica ao top e o bottom. E o segundo valor ao left e right.
● Para 3 valores: o primeiro se aplica ao top, o segundo ao right e left e o terceiro ao bottom.
● Para 4 valores, a ordem é: top, right, bottom e left (sentido horário).
